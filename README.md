# async-fifo-rtl-to-gds-synopsys
ASIC project demonstrating Asynchronous FIFO architecture, CDC verification, synthesis, STA, and physical design using Synopsys tools.

This project implements and verifies an Asynchronous FIFO used for safe data transfer between independent clock domains. The design incorporates Gray-code pointer synchronization, full/empty detection logic, and CDC-safe architecture.

The project demonstrates the complete ASIC design flow from RTL development and functional verification to synthesis, physical implementation, timing analysis, and GDS generation using Synopsys EDA tools.

## Key Skills Demonstrated
- Verilog RTL Design
- Functional Verification
- Clock Domain Crossing (CDC)
- FIFO Architecture
- Design Compiler (DC)
- IC Compiler II (ICC2)
- PrimeTime STA
- ASIC RTL-to-GDS Flow
